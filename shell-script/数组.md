# 数组
用括号来表示数组，数组元素用"空格"符号分割开
array=(v1 v2 v3 v4)

读取某个元素: ${数组名[下标]}

使用 @ 符号可以获取数组中的所有元素: ${array[@]}

## 数组长度
我们要获取数组的长度，**我们需要使用 @ 或者 * 将数组扩展成列表，然后再次使用 # 获取数组的长度。** 
```
${#array_name[@]}
${#array_name[*]}
```

案例
```
-> % array=(a b c d e f)
-> % echo ${array[3]}       
d
-> % echo ${array[@]}       
a b c d e f
-> % echo ${#array[@]}
6
-> % echo ${#array[*]}
6
```
